Si estás recibiendo un error prolongado, es posible que tu conexión a internet no sea estable. Intenta cambiar entre Wi-Fi y datos móviles (4G) o prueba nuevamente desde una ubicación diferente.

### "An internal error has occurred"

Prueba estos pasos para resolver el problema:

1. Cambia entre Wi-Fi y datos móviles (4G). Si el error persiste,
2. Reinicia tu teléfono. Si el error persiste,
3. Ve a **Acerca de -> Configurar Android 11 Setup**. Si el error persiste,
4. Reinstala la aplicación **translatefgo**. Si el error aún persiste,
5. Reporta el problema como un error en **Reportar Error**.

### Shizuku (El permiso de ADB está limitado)

Este tipos de problemas puede darse en lo siguientes dispositivos y sistemas:

#### MIUI (Xiaomi, POCO)

1. Habilita la opción **"Depuración USB (Opciones de seguridad)"** en **"Opciones de desarrollador"**. Ten en cuenta que esta es una opción separada de **"Depuración USB"**.

#### ColorOS (OPPO & OnePlus)

1. Desactiva **"Monitoreo de permisos"** en **"Opciones de desarrollador"**.

#### Flyme (Meizu)

1. Desactiva **"Protección de pagos Flyme"** en **"Opciones de desarrollador"**.
