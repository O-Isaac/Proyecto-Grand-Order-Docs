# Actualizar
La aplicación recibirá actualizaciones periódicamente. Mientras mantenga la aplicación instalada, verifíquela regularmente.

<Callout type="info" emoji="⚒️">
 En el apartado del Servidor seleccionado, el estado de **«Instalado»** en el paquete, cambiará a **«Actualización Disponible»** en caso de haber una.
</Callout>

Para actualizar el nuevo paquete debe hacer lo siguiente:
- Asegúrese de tener instalado la última versión de Fate/Grand Order.
- Inicie la aplicación de Fate/Grand Order para asegurarse de que no haya ninguna actualización adicional disponible. Una vez hecho eso, cierre la aplicación.
- En la aplicación de Rayshift.io Translate, presione «Actualizar» en el apartado del Servidor seleccionado.
- Una vez actualizado, reabrir la aplicación de Fate/Grand Order.

<Callout>
La aplicación también se actualizará automáticamente en caso de haber modificaciones menores, siempre que su dispositivo sea compatible. 
Puede desactivar esta opción en el apartado de **"About"**
</Callout>
