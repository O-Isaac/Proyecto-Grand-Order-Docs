# Desinstalación
Si usted desea desinstalar el parche solo tiene que seguir estos pasos:
* Cerrar la aplicación de Fate/Grand Order.
* Abrir la aplicación de Rayshift.io Translate, presionar el botón de «Desinstalar».
* Reabrir la aplicación de Fate/Grand Order.
* También puede borrar el caché del juego en caso de no poder utilizar la aplicación
