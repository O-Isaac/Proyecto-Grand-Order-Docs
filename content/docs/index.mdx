## Instalación

**Compatibilidad:** Android 7–15+  
**Root:** No requerido  
**Nota:** En algunos dispositivos modernos puede ser necesario Shizuku.

### Pasos

1. **Instala Fate/Grand Order (versión JP)** y asegúrate de tener la última versión disponible.
2. **Abre el juego** y accede a tu cuenta para asegurarte de que los datos estén actualizados.  
   Si aparece una solicitud de descarga adicional, acéptala. Luego **cierra la aplicación**.
3. **Descarga e instala el APK más reciente** desde la [página de lanzamientos](#).
4. **Configura la aplicación** siguiendo las instrucciones proporcionadas.  
   Consulta el video tutorial si necesitas ayuda.
5. En la app, **haz clic en “Install”** debajo del _bundle_ que desees instalar.  
   Solo es necesario instalar **un bundle a la vez**.
6. **Vuelve a abrir Fate/Grand Order.**

## Actualizaciones

Los scripts se actualizan de forma regular. Por favor, mantén la aplicación instalada y revísala con frecuencia:  
el estado de "Instalado" cambiará a **"Actualización disponible"** cuando haya una nueva versión.

### Para actualizar después de que se añada nuevo contenido:

1. **Abre Fate/Grand Order** y acepta la descarga de datos.
2. **Cierra Fate/Grand Order** y haz clic en **"Update"** dentro de la aplicación de traducción.
3. **Vuelve a abrir Fate/Grand Order.**

La aplicación también puede **actualizar automáticamente** los scripts instalados cuando se detectan cambios menores, siempre que tu dispositivo sea compatible.  
Esta opción puede desactivarse desde la pestaña **Acerca de (About)**.

## Desinstalación

1. **Cierra Fate/Grand Order.**
2. **Abre la aplicación** y haz clic en **"Uninstall"**.
3. **Vuelve a abrir Fate/Grand Order.**

## Solución de Problemas

Si tienes errores durante el uso de la aplicación o del juego, aquí encontrarás las soluciones más comunes.

Sigue las instrucciones detalladas en cada sección antes de solicitar ayuda.

<Accordions type="single">
<Accordion title="Problemas de conexión o errores largos">
Si ves un error muy extenso, es posible que tu conexión a internet no sea estable.  
Intenta cambiar entre Wi-Fi y datos móviles (4G), o prueba en una ubicación diferente.
</Accordion>

<Accordion title="Errores como 'Asset database is corrupt' o 'Failed to read beyond the end of the stream'">
Reinicia tu teléfono. Si el problema persiste, cambia el **modo de acceso al
sistema de archivos** en la pestaña **Acerca de (About)**.

- Para dispositivos con Android 12 o inferior, selecciona **SAF**.
- Para dispositivos más recientes, usa **Shizuku**.

</Accordion>

<Accordion title="Error 'An internal error has occurred'">

1. Cambia entre Wi-Fi y datos móviles.
2. Reinicia tu teléfono.
3. Ve a **About → Configure Android 11 Setup**.
4. Reinstala la aplicación TranslateFGO. 5. Si el error continúa, repórtalo como
   un bug en GitHub.

</Accordion>
